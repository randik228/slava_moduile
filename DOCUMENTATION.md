# Модуль matrix_gf2

Библиотека для работы с матрицами над конечными полями Галуа GF(p^m) на C++17.

## Возможности

### Элементы поля Галуа
- Поддержка полей GF(p^m), где p - простое число, m - степень расширения
- Арифметические операции: сложение, вычитание, умножение, деление
- Нахождение обратных элементов
- Поддержка полей GF(2), GF(3), GF(5) и других

### Матрицы
- **Создание матриц**: нулевые, единичные, случайные, из массивов
- **Доступ к элементам**: чтение и запись элементов
- **Арифметические операции**:
  - Сложение и вычитание матриц
  - Умножение матрица×матрица
  - Умножение матрица×вектор
  - Умножение на скаляр
  - Транспонирование

### Метод Гаусса
- **Прямой ход Гаусса**: приведение к ступенчатому виду
- **Обратный ход Гаусса**: приведение к систематическому виду
- **RREF (Reduced Row Echelon Form)**: приведённый ступенчатый вид
- **Вычисление ранга матрицы**
- **Поиск обратимой подматрицы**

### Обратные матрицы
- Проверка обратимости матрицы
- Вычисление обратной матрицы методом Гаусса-Жордана
- Поиск обратимых подматриц

### Учебный режим
- Пошаговый вывод элементарных преобразований строк
- Объяснение целей каждого шага
- Детальное отслеживание процесса решения

## Примеры использования

### Базовые операции

```cpp
#include "matrix_gf2/matrix.hpp"
using namespace matrix_gf2;

// Создание матрицы 3x3 над GF(2)
Matrix A({{1, 0, 1},
          {0, 1, 1},
          {1, 1, 0}}, 2, 1);

// Единичная матрица
Matrix I = Matrix::identity(3, 2, 1);

// Сложение
Matrix C = A + I;

// Умножение
Matrix D = A * I;

// Транспонирование
Matrix AT = A.transpose();

// Ранг
size_t rank = A.rank();
```

### Метод Гаусса

```cpp
// Приведение к ступенчатому виду
auto result = A.forwardGauss(false);
std::cout << "Ранг: " << result.rank << "\n";
std::cout << "Матрица:\n" << result.matrix << "\n";

// Приведение к RREF
result = A.reducedRowEchelonForm(false);
std::cout << "RREF:\n" << result.matrix << "\n";
```

### Обратная матрица

```cpp
// Проверка обратимости
if (A.isInvertible()) {
    auto invA = A.inverse(false);
    if (invA) {
        std::cout << "A^(-1):\n" << *invA << "\n";
        
        // Проверка
        Matrix check = A * (*invA);
        std::cout << "A * A^(-1):\n" << check << "\n";
    }
}
```

### Учебный режим

```cpp
// Включить пошаговые объяснения
auto result = A.reducedRowEchelonForm(true);

// Вывести все шаги
for (const auto& step : result.steps) {
    std::cout << step << "\n";
}

// Нахождение обратной с объяснениями
auto inv = A.inverse(true);
```

### Работа с GF(3)

```cpp
// Матрица над GF(3)
Matrix A({{1, 2, 0},
          {0, 1, 2},
          {2, 0, 1}}, 3, 1);

std::cout << "Ранг: " << A.rank() << "\n";

if (A.isInvertible()) {
    auto inv = A.inverse(false);
    std::cout << "Обратная:\n" << *inv << "\n";
}
```

### Решение систем линейных уравнений

```cpp
// Система уравнений над GF(2):
//   x + z = 1
//   y + z = 0
//   x + y = 1

Matrix sys({{1, 0, 1, 1},
            {0, 1, 1, 0},
            {1, 1, 0, 1}}, 2, 1);

auto result = sys.reducedRowEchelonForm(false);

std::cout << "Решение:\n";
std::cout << "x = " << result.matrix(0, 3) << "\n";
std::cout << "y = " << result.matrix(1, 3) << "\n";
std::cout << "z = " << result.matrix(2, 3) << "\n";
```

## API документация

### Класс GFElement

Представляет элемент поля Галуа GF(p^m).

**Конструкторы:**
- `GFElement(uint32_t p, uint32_t m, const std::vector<uint32_t>& modulus)` - нулевой элемент
- `GFElement(uint32_t value, uint32_t p, uint32_t m, ...)` - из целого числа
- `GFElement(const std::vector<uint32_t>& coeffs, ...)` - из коэффициентов полинома

**Операции:**
- `operator+, -, *, /` - арифметические операции
- `inverse()` - обратный элемент
- `isZero()`, `isOne()` - проверки
- `getValue()` - получение значения

### Класс Matrix

Представляет матрицу над полем GF(p^m).

**Конструкторы:**
- `Matrix(size_t rows, size_t cols, uint32_t p, uint32_t m, ...)` - нулевая матрица
- `Matrix(const std::vector<std::vector<uint32_t>>& data, ...)` - из массива
- `Matrix::identity(size_t n, ...)` - единичная матрица
- `Matrix::zero(size_t rows, size_t cols, ...)` - нулевая матрица
- `Matrix::random(size_t rows, size_t cols, ...)` - случайная матрица

**Доступ к элементам:**
- `at(size_t i, size_t j)` - доступ с проверкой границ
- `operator()(size_t i, size_t j)` - доступ без проверки

**Операции:**
- `operator+, -, *` - арифметические операции
- `operator*(const std::vector<GFElement>&)` - умножение на вектор
- `transpose()` - транспонирование
- `getRow(i)`, `getCol(j)` - получение строки/столбца
- `swapRows(i, j)` - обмен строк
- `multiplyRow(i, scalar)` - умножение строки
- `addRow(dest, src, scalar)` - добавление строки

**Метод Гаусса:**
- `forwardGauss(bool educational)` - прямой ход
- `backwardGauss(bool educational)` - обратный ход
- `reducedRowEchelonForm(bool educational)` - RREF
- `rank()` - вычисление ранга

**Обратные матрицы:**
- `isInvertible()` - проверка обратимости
- `inverse(bool educational)` - вычисление обратной
- `findInvertibleSubmatrix()` - поиск обратимой подматрицы
- `submatrix(rows, cols)` - извлечение подматрицы

## Особенности реализации

### Поля Галуа
- Элементы представлены как полиномы с коэффициентами из GF(p)
- Операции выполняются по модулю неприводимого полинома
- Поддержка расширенного алгоритма Евклида для нахождения обратных элементов

### Метод Гаусса
- Поиск ведущих элементов по столбцам
- Нормализация строк (ведущий элемент = 1)
- Обнуление элементов выше и ниже ведущих
- Отслеживание ведущих столбцов для вычисления ранга

### Учебный режим
- Детальное описание каждого шага
- Объяснение целей преобразований
- Вывод промежуточных результатов
