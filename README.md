# slava_moduile - Matrix GF(p^m) Library

Библиотека для работы с матрицами над конечными полями Галуа GF(p^m) на C++17.

## Возможности

✅ Матрицы над GF(p^m) - создание, доступ к элементам, чтение/запись  
✅ Операции: умножение матрица×вектор, матрица×матрица, транспонирование  
✅ Прямой и обратный гаусс, приведение к ступенчатому и систематическому виду  
✅ Поиск ранга, обратимой подматрицы  
✅ Построение обратимой матрицы, проверка обратимости  
✅ Учебный режим: вывод элементарных преобразований строк и объяснений  

## Быстрый старт

```bash
# Сборка
mkdir build && cd build
cmake ..
cmake --build .

# Запуск тестов
./test_matrix_gf2

# Запуск примеров
./example_basic
./example_educational
```

## Примеры

```cpp
#include "matrix_gf2/matrix.hpp"
using namespace matrix_gf2;

// Создание матрицы над GF(2)
Matrix A({{1, 0, 1},
          {0, 1, 1},
          {1, 1, 1}}, 2, 1);

// Метод Гаусса с учебным режимом
auto result = A.reducedRowEchelonForm(true);

// Обратная матрица
auto inv = A.inverse(true);
```
