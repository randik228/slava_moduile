# Matrix GF(p^m) Module - Implementation Summary

## Обзор

Полная реализация модуля `matrix_gf2` для работы с матрицами над конечными полями Галуа GF(p^m) на языке C++ с использованием стандарта C++17.

## Выполненные требования

### ✅ 1. Матрицы над GF(p^m)
- **Создание**: Нулевые, единичные, случайные матрицы, создание из массивов
- **Доступ к элементам**: Безопасный доступ через `at()` и быстрый через `operator()`
- **Чтение/запись**: Полная поддержка чтения и изменения элементов

### ✅ 2. Арифметические операции
- **Матрица × вектор**: Эффективное умножение с проверкой размерностей
- **Матрица × матрица**: Полная реализация матричного умножения
- **Транспонирование**: Создание транспонированной матрицы
- **Сложение/вычитание**: Поэлементные операции

### ✅ 3. Метод Гаусса
- **Прямой ход**: Приведение к ступенчатому виду (REF)
- **Обратный ход**: Приведение к приведённому ступенчатому виду (RREF)
- **Поиск ранга**: Вычисление ранга матрицы
- **Обратимая подматрица**: Поиск максимальной обратимой подматрицы

### ✅ 4. Обратные матрицы
- **Проверка обратимости**: Быстрая проверка через вычисление ранга
- **Построение обратной**: Метод Гаусса-Жордана для нахождения обратной матрицы
- **Верификация**: Автоматическая проверка результата

### ✅ 5. Учебный режим
- **Пошаговый вывод**: Детальное описание каждого шага преобразований
- **Объяснения**: Понятные комментарии о целях каждой операции
- **Промежуточные результаты**: Вывод матрицы после каждого важного шага

### Ключевые классы

#### GFElement
Представляет элемент поля Галуа GF(p^m):
- Хранение как полином с коэффициентами из GF(p)
- Арифметические операции с автоматическим приведением по модулю
- Поиск обратных элементов через расширенный алгоритм Евклида

#### Matrix
Представляет матрицу над GF(p^m):
- Двумерный массив элементов GFElement
- Полный набор матричных операций
- Методы Гаусса с поддержкой учебного режима

#### GaussResult
Структура результата Гауссова исключения:
- Преобразованная матрица
- Ранг матрицы
- Индексы ведущих столбцов
- Шаги преобразования (для учебного режима)

## Примеры использования

### Базовые операции
```cpp
Matrix A({{1, 0, 1},
          {0, 1, 1},
          {1, 1, 0}}, 2, 1);  // Матрица над GF(2)

Matrix B = Matrix::identity(3, 2, 1);
Matrix C = A * B;
Matrix AT = A.transpose();
size_t rank = A.rank();
```

### Метод Гаусса
```cpp
auto result = A.forwardGauss(false);
std::cout << "Ранг: " << result.rank << "\n";

result = A.reducedRowEchelonForm(false);
std::cout << "RREF:\n" << result.matrix << "\n";
```

### Обратная матрица
```cpp
if (A.isInvertible()) {
    auto inv = A.inverse(false);
    Matrix check = A * (*inv);  // Должна быть единичной
}
```

### Учебный режим
```cpp
// Включить пошаговые объяснения
auto result = A.reducedRowEchelonForm(true);
for (const auto& step : result.steps) {
    std::cout << step << "\n";
}
```

## Производительность

### Сложность алгоритмов
- Сложение/вычитание: O(n×m)
- Умножение матриц: O(n×m×k)
- Транспонирование: O(n×m)
- Метод Гаусса: O(n²×m)
- Обратная матрица: O(n³)
